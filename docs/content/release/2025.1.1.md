我们很高兴地宣布 DataCap 2025.1.1 版本正式发布！本次更新专注于提升系统稳定性、修复关键问题，并新增了多项实用功能。

## 📅 发布信息

- **版本号**: 2025.1.1
- **发布日期**: 2025年7月28日
- **项目地址**: https://github.com/devlive-community/datacap
- **官方网站**: https://datacap.devlive.org/

## 🎯 版本亮点

### 🆕 新功能特性

- **🐬 DolphinDB 数据库支持**: 全新支持 DolphinDB 时序数据库，扩展了高性能时间序列数据处理能力
- **📊 数据集通知系统**: 新增数据集通知配置功能，支持自定义通知规则和异步任务处理
- **🔧 Flink SQL 解析器**: 增强了 SQL 解析能力，现已支持 Flink SQL 语法解析
- **📱 多 SQL 语句支持**: 优化解析器，支持在单次操作中处理多条 SQL 语句
- **🔌 插件详情展示**: 改进插件商店界面，提供更详细的插件信息展示

### 🛠️ 重要修复

- **数据导出格式化**: 修复了数据导出时未指定格式化的问题，确保导出数据的正确性
- **Doris 插件优化**: 解决了 Doris 插件在渲染数据库名称时的显示异常 (#943)
- **分页国际化**: 完善了分页组件的多语言支持，提升用户体验
- **JSON 转换精度**: 修复了 JSON 转换过程中 LocalDateTime 丢失秒精度的问题 (#920)
- **数据重复问题**: 解决了数据集执行后可能导致数据重复的问题
- **用户密码安全**: 修复了管理员修改用户密码时可能导致密码明文存储的安全隐患

### 🔧 系统优化

- **达梦数据库增强**: 改进了达梦数据库连接器的兼容性，解决了 Docker 平台兼容性问题
- **通知中心升级**: 优化了通知中心的界面和功能，提供更好的消息管理体验
- **代码质量提升**: 修复了多个静态代码分析发现的潜在问题，提高了代码稳定性
- **Docker 镜像优化**: 支持构建多平台 Docker 镜像，提升部署灵活性

### 🗑️ 功能移除

- **Pipeline 功能清理**: 移除了 Pipeline 相关功能及代码，简化了系统架构
- **无效组件清理**: 清理了未使用的属性和无效配置，优化了系统性能

## 🚀 升级指南

1. **备份数据**: 升级前请务必备份您的数据和配置文件
2. **停止服务**: 停止当前运行的 DataCap 服务
3. **更新版本**: 下载最新的 2025.1.1 版本文件
4. **数据库升级**: 如有数据库结构变更，请参考升级脚本
5. **重启服务**: 启动新版本服务并验证功能

## 📝 注意事项

- 本版本移除了 Pipeline 功能，如果您在使用相关功能，请在升级前做好数据迁移
- 新增的 DolphinDB 支持需要相应的插件配置
- 建议在生产环境升级前，先在测试环境进行完整验证
